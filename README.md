# 🖥️ Locapic - Backend

Backend de l’application mobile Locapic, développé avec Node.js, Express et MongoDB. Il gère le stockage des lieux favoris par utilisateur (identifié par un pseudo).

## ⚙️ Technologies

- Node.js
- Express
- MongoDB + Mongoose
- Déployé sur Vercel
